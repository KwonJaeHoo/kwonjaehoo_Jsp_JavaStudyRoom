## 홈페이지 기능

* 2계층     
  * 관리자 / 사용자     

* 회원가입 / 로그인 / 로그아웃    

* 마이페이지     
  * 내 정보 수정     
  * 내가 쓴 글 / 리스트, 삭제, 전체삭제
  * 회원탈퇴          

* 게시판
  * 리스트
  * 검색기능
  * 정렬 - 최신/추천/조회    

* 글 작성
  * 비밀글 설정하기
  
* 게시글 상세페이지 
  * 글 추천하기    
  * 글 삭제하기
